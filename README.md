# KBar 3 中文文档

这是 KBar 3 的中文文档网站，基于 VitePress 构建。

## 🌟 特性

- 完整的中文文档
- 响应式设计
- 暗色/亮色主题支持
- 优化的搜索功能

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建静态文件
npm run docs:build

# 预览构建结果
npm run docs:preview
```
